Git

  • git clone <ссылка с GitHub>- расшифровывается как "клонирование с удаленного репозитория на компьютер"
  • git status - расшифровывается как "просмотр изменений"
  • git add - расшифровывается как "добавить файлы в отслеживаемые"
  • git commit -m "fircs commit" - расшифровывается как "сделать первый коммит"
  • git push - расфшифровывается как "внести изменения в удаленный репозиторий"
  • git log - расфшифровывается как "проверить изменения через терминал"
  • git pull - расфшифровывается как "внести изменения из удаленного репозитория в локальный"
  • git branch - расфшифровывается как "создать ветку в локальном репозитории"

Git

  • clone <ссылка с GitHub>- расшифровывается как "клонирование с удаленного репозитория на компьютер"
  • status - расшифровывается как "просмотр изменений"
  • add - расшифровывается как "добавить файлы в отслеживаемые"
  • commit -m "fircs commit" - расшифровывается как "сделать первый коммит"
  • push - расфшифровывается как "внести изменения в удаленный репозиторий"
  • log - расфшифровывается как "проверить изменения через терминал"
  • pull - расфшифровывается как "внести изменения из удаленного репозитория в локальный"
  • branch - расфшифровывается как "создать ветку в локальном репозитории"
